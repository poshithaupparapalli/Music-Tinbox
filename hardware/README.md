# Hardware

KiCad schematic and PCB files go here.

## Options

### Option A: Use Sage's design as-is (recommended for first build)
1. Copy her KiCad files from `boardwalk/music_tinbox/` into this folder
2. Open in KiCad, review, generate gerbers
3. Upload gerbers to JLCPCB

### Option B: Modify her design
1. Copy her files, then edit in KiCad
2. More learning, but more risk of errors
3. Only do this if you have extra time

### Option C: Design from scratch using lcamtuf's schematic as reference
1. Maximum learning
2. Not recommended on a 2-week timeline with no prior experience
3. Cool future project though
